### 🧩 Claude Code plugins

My own [marketplace](https://github.com/yoannyviquel/marketplace) — install with:

```bash
/plugin marketplace add yoannyviquel/marketplace
```

| Plugin | What it does |
|--------|--------------|
| 🧠 **memory** | Local SQLite + MCP persistent memory — auto-recall across sessions |
| 🔷 **microsoft-tfs** | Azure DevOps / TFS from the terminal — PRs, builds, releases, work items |
| 📊 **consumption-status-bar** | Live token / usage status line |
| 🤖 **agents** | Dev agents per stack (.NET, React, Node, iOS) |
